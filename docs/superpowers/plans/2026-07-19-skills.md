# Skills Implementation Plan

> **For agentic workers:** execute task-by-task, TDD. Steps use `- [ ]`. Spec:
> `docs/superpowers/specs/2026-07-19-skills-design.md`.

**Goal:** First-class skills ÔÇö authored from files/inline config, stored + versioned in rbrun's DB,
staged into the sandbox per turn, with divergence surfaced (warn Ôćĺ Cancel/Reload).

**Architecture:** Engine-only. `SkillArchive` (folderÔçäblob) ┬Ě `Skill`/`SkillVersion` (content-addressed,
tenant-scoped) ┬Ě `SkillSeeder` (compare, never clobber) ┬Ě `AgentTurn` materializes current versions Ôćĺ
`run(skills: dir)` ┬Ě `ReloadSkills` tool ┬Ě Skills panel in the footer user dropdown.

**Tech:** Rails engine, ActiveRecord, `Gem::Package`/`zlib`, minitest (DI, no mocks), ViewComponent.

## Global Constraints (from spec)

- **Agent stages skills from the DB (current `SkillVersion`) ÔÇö never files/config at turn time.**
- File/inline = seed sources only. Seeder never clobbers: version + warn on diff; resolve Cancel/Reload.
- Always-on tenancy; skills/versions tenant-scoped. No registry. Pure sub-gems untouched.

### Task 4 ÔÇö `Rbrun::SkillSeeder` (compare, never clobber)

**Files:** Create `app/services/rbrun/skill_seeder.rb`; Test `test/services/rbrun/skill_seeder_test.rb`.

**Consumes:** `SkillArchive`, `Skill`, `config`. **Produces:** `SkillSeeder.new(tenant:, authored:).call -> [results]` where `authored` = the merged file-glob + inline list of `{slug, name, files}`; `results` carry `:created | :unchanged | :diverged | :issue`.

- [ ] Authored assembly: `.from_config(config, tenant:)` globs `skills_path/*/SKILL.md` (dir name = slug, read the folder's files) + `config.skills`. A malformed entry (no `SKILL.md`, pack raises) Ôćĺ `:issue`.
- [ ] Per authored skill: `digest = SkillArchive.digest_files(files)`; find `Skill` by `[tenant, slug]`:
  - none Ôćĺ `Skill.create!` + `promote!(digest, archive, source)` Ôćĺ `:created`.
  - `digest Ôłł {current.digest, dismissed_digest}` Ôćĺ clear `divergence_digest` if set Ôćĺ `:unchanged`.
  - else Ôćĺ `update!(divergence_digest: digest)` (re-pack at reconcile time) Ôćĺ `:diverged`. **Never** touches `current`.
- [ ] Test (DI, no boot): create a Skill+version; feed identical authored Ôćĺ `:unchanged`; changed body Ôćĺ `:diverged` + `current` unchanged + `divergence_digest` set; a body matching `dismissed_digest` Ôćĺ `:unchanged` (no re-warn); new slug Ôćĺ `:created`; a files-hash without `SKILL.md` Ôćĺ `:issue`. rubocop. Commit.

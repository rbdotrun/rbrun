# Skills Implementation Plan

> **For agentic workers:** execute task-by-task, TDD. Steps use `- [ ]`. Spec:
> `docs/superpowers/specs/2026-07-19-skills-design.md`.

**Goal:** First-class skills ‚Äî authored from files/inline config, stored + versioned in rbrun's DB,
staged into the sandbox per turn, with divergence surfaced (warn ‚Üí Cancel/Reload).

**Architecture:** Engine-only. `SkillArchive` (folder‚áÑblob) ¬∑ `Skill`/`SkillVersion` (content-addressed,
tenant-scoped) ¬∑ `SkillSeeder` (compare, never clobber) ¬∑ `AgentTurn` materializes current versions ‚Üí
`run(skills: dir)` ¬∑ `ReloadSkills` tool ¬∑ Skills panel in the footer user dropdown.

**Tech:** Rails engine, ActiveRecord, `Gem::Package`/`zlib`, minitest (DI, no mocks), ViewComponent.

## Global Constraints (from spec)
- **Agent stages skills from the DB (current `SkillVersion`) ‚Äî never files/config at turn time.**
- File/inline = seed sources only. Seeder never clobbers: version + warn on diff; resolve Cancel/Reload.
- Always-on tenancy; skills/versions tenant-scoped. No registry. Pure sub-gems untouched.

### Task 4 ‚Äî `Rbrun::SkillSeeder` (compare, never clobber)

**Files:** Create `app/services/rbrun/skill_seeder.rb`; Test `test/services/rbrun/skill_seeder_test.rb`.

**Consumes:** `SkillArchive`, `Skill`, `config`. **Produces:** `SkillSeeder.new(tenant:, authored:).call -> [results]` where `authored` = the merged file-glob + inline list of `{slug, name, files}`; `results` carry `:created | :unchanged | :diverged | :issue`.

- [ ] Authored assembly: `.from_config(config, tenant:)` globs `skills_path/*/SKILL.md` (dir name = slug, read the folder's files) + `config.skills`. A malformed entry (no `SKILL.md`, pack raises) ‚Üí `:issue`.
- [ ] Per authored skill: `digest = SkillArchive.digest_files(files)`; find `Skill` by `[tenant, slug]`:
  - none ‚Üí `Skill.create!` + `promote!(digest, archive, source)` ‚Üí `:created`.
  - `current.digest == digest` ‚Üí clear `divergence_digest` if set ‚Üí `:unchanged`.
  - else ‚Üí `update!(divergence_digest: digest)` (store the packed archive on the flag? no ‚Äî re-pack at reconcile) ‚Üí `:diverged`. **Never** touches `current`.
- [ ] Test (DI, no boot): create a Skill+version; feed identical authored ‚Üí `:unchanged`; changed body ‚Üí `:diverged` + `current` unchanged + `divergence_digest` set; new slug ‚Üí `:created`; a files-hash without `SKILL.md` ‚Üí `:issue`. rubocop. Commit.
